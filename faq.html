FAQ

How to use this FAQ?
Just like that!
How to get points?
You earn points by tagging and so on...
How do I start spraying?
Push icon in home screen
What "Bust" means?
A player from rival gang caught you in spraying, therefore your spraying event has been cancelled
What "Untagged" means?
None of the Gangs have been tagged this location
How to set my name?
1. Go to "tweaks page",
2. Write your name in "Set Name" input field
3. Then push "Save"
How to set my mood?
1. Go to "tweaks page",
2. Write your name in "Set Mood" input field
3. Then push "Save"
How to log out?
1. Go to "tweaks page",
2. Push "Logout"
3. Then push "Yes"
